# What is Spring?

Spring is a Java framework that helps developers build applications with less boilerplate code. It manages objects, dependencies, and provides many modules for building enterprise applications.

## Why Spring?

- Reduces boilerplate code
- Promotes loose coupling
- Easier testing
- Scalable and maintainable

## Key Features

- IoC (Inversion of Control)
- Dependency Injection
- Spring MVC
- Spring Data JPA
- Spring Security

## Example

Without Spring:
```java
Laptop laptop = new Laptop();
Alien alien = new Alien(laptop);
```

With Spring:
The Spring container creates and injects the required objects.
