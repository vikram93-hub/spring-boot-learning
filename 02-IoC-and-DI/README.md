# IoC (Inversion of Control) and Dependency Injection(DI)

## What is IoC?

Inversion of Control (IoC) is a principle where the Spring Framework manages the creation and lifecycle of objects instead of the programmer creating them manually using the `new` keyword.

## What is Dependency Injection?

Dependency Injection (DI) is a technique where Spring automatically provides the required dependencies to a class.

## Benefits

- Loose coupling
- Easier testing
- Better maintainability
- Reusable components

## Key Annotations

- @Component
- @Configuration
- @ComponentScan
- @Autowired

## What I Learned

- Spring creates and manages beans.
- Dependencies can be injected automatically.
- The IoC container reduces object coupling.
