# Spring Boot Web

## Objective

Learn how to build web applications and REST APIs using Spring Boot.

## What is Spring Boot Web?

Spring Boot Web is a module that helps developers create web applications without configuring an external web server.

It uses an embedded Tomcat server by default.

## Features

- Embedded Tomcat Server
- REST API Development
- MVC Architecture Support
- Easy Request Handling

## Dependency Used

spring-boot-starter-web

## Common Annotations

### @Controller

Used to create traditional web applications.

### @RestController

Used to create REST APIs.

### @RequestMapping

Maps HTTP requests to controller methods.

### @GetMapping

Handles HTTP GET requests.

## Advantages

- Quick web application development
- No external server configuration
- Easy REST API creation
- Minimal configuration
